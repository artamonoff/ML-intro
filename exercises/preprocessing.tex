\section{Preprocessing}

\textbf{Замечание}: \textit{рассмотрите следующие преобразования переменных:
\begin{itemize}
	\item квантильное (для гауссового распределения)
	\item Box-Cox
	\item Yeo-Johnson
\end{itemize}
}

\begin{exercise}
Загрузите датасет \texttt{Labour}
\begin{enumerate}
	\item Нарисуйте гистограммы для каждой переменной в уровнях и после стандартных преобразований
	\item Нарисуйте диаграммы рассеяния в уровнях и после стандартных преобразований
\end{enumerate}
\end{exercise}

	\item Нарисуйте диаграммы рассеяния в уровнях и после стандартных преобразований
\end{enumerate}
\end{exercise}
